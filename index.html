The problem

What stateless agents can't do

Behavioral consistency in current LLM agents is an artifact of the context window, not of the architecture. Three structural gaps follow from that.

01

No accumulating pressure

Content that can't be said yet doesn't exist for the model. There is nothing building toward a break in the discourse.

02

No structural history

The agent in turn fifty is not systematically different from the agent in turn one. Nothing about the interaction accumulates.

03

No reproducibility

Same state, same question, different behavioral trajectory. Behavior that can't be replayed can't be audited.

Architecture

Five layers, one interface

Each layer has exactly one job. State moves through in one direction — the model never writes back into the engine directly.

01

Psychodynamic engine 0 model calls

Seven state variables update by deterministic equations each conversational turn: drive, resistance, anxiety, guilt, jouissance, unconscious pressure, defensive rigidity.

02

LLM coupling protocol 1 model call / turn

The engine writes a precise, structured instruction. The model's only job is to render that instruction as natural language in a given voice.

03

Deterministic signal extractor 0 model calls

Structural regex, weighted keyword matching, and cosine similarity read the rendered response back into signal — no classifier model in the loop.

04

Social influence & discursive layer population-scale

When a subject sits inside a population, five factors modulate its signals: authority, consensus, group influence, source trust, reputational cost. A Lacanian discursive-position matrix (master, university, hysteric, analyst) adds a qualitative axis of social bond — how an agent relates to the Other, not just how it feels.

05

Deterministic linguistic layer 0 model calls

LIWC particles and enunciation deixis, counted — not inferred. Yields an auditable lexical signature of psychodynamic state that detects structure from observable text alone (AUC 0.82 solo, 0.85 in ensemble).

Multi-agent safety

A parametric model organism for collusion

Because propensity is an equation, not a trained policy, ANIMA can't sandbag — it can't recognize an evaluation and fake alignment. A testbed that cannot perform is a methodological asset when the field doubts whether its evals measure alignment or acting.
